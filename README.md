## Demos
https://localnerve.github.io/ui-elements  
NOTE: Some of these may only exhibit behavior on small devices or touch screens.

## How To Run
Requires NodeJS to run the demos.
  1. Clone repo
  2. `npm install`
  3. `npm start`
  4. Navigate to localhost:3010
